# Public Release Notes

这个文件记录公开版仓库相对内部工作副本做过的整理。

## 已完成调整

- 删除 `.DS_Store`
- 新增 `LICENSE`
- 新增 `.gitignore`
- 移除内部合同审批模板
- 清理模板中的明文密钥、Webhook、固定实例地址和默认业务配置
- 将模板文件名统一为英文 `kebab-case`
- 将公开模板按 `core`、`integration`、`experimental` 分类说明

## 公开版范围

公开版保留：

- skill 主说明
- 参考资料
- 本地校验脚本
- 公开模板库

公开版不保留：

- 明显偏内部审批流程的模板
- 依赖内部默认实例地址或内置业务口径的配置

## 使用提醒

- 模型名只是示例值，请替换为你自己的可用模型。
- 集成类模板需要用户自行填写 API 地址、变量和鉴权配置。
- 如果模板与目标 FastGPT 版本不完全一致，应优先以导出模板或实际运行环境为准。
