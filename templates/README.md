- `sales-coaching-master.json`

- `ai-finance-daily.json`
- `invoice-recognition-assistant.json`
- `resume-screening-assistant-excel.json`
- `interview-assistance-expert.json`
- `it-operations-qa-assistant.json`

- `travel-budget-monitor-assistant.json`

这些模板更适合做结构参考或二次改造：

- `auto-sales-script-recommendation-assistant.json`
- `learning-plan-book.json`

## Public Edition Rules

- 公开版不包含内部业务审批模板。
- `integration` 与 `experimental` 模板不承诺开箱即用。
- 选择基底模板时，应检查整个目录，而不是只看最初的锚点模板。
